# Syntecxhub-Image-Slider
Responsive Image Slider using HTML, CSS and JavaScript.
A responsive **Image Slider** built using **HTML, CSS, and JavaScript**.
This project allows users to navigate images using **next/previous buttons, dots navigation, keyboard arrows, and auto sliding**.

---

## 🚀 Features

* Next / Previous navigation buttons
* Dots indicator navigation
* Auto slide every few seconds
* Pause slider on hover
* Keyboard navigation (Arrow Left / Arrow Right)
* Smooth fade animation
* Responsive design for mobile and desktop

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**

## 🎮 How It Works

* Images are displayed one at a time.
* JavaScript controls the **current slide index**.
* Navigation buttons, dots, and keyboard events update the slide index.
* The `active` class determines which slide is visible.
